## Ejecución local
1. Es necesario crear un archivo .env de la siguiente forma:
```
NEXT_PUBLIC_SUPABASE_URL=<>
NEXT_PUBLIC_SUPABASE_ANON_KEY=<>
```


2. Se instalan las dependencias:
```
npm i
```

3. Se ejecuta el siguiente código:
```
npm run dev
```
